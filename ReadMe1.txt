USB Stealth Screenshot Capture Tool
====================================

This USB drive will automatically capture screenshots when inserted into a computer.

How it works:
1. Insert USB into target computer
2. Tool waits 10 minutes (avoids suspicion)
3. Takes screenshots every 30 seconds
4. Saves screenshots to hidden folder on USB

Files included:
- autorun.inf: Windows AutoRun configuration
- launcher.bat: Main launcher script
- stealth_capture.py: Python screenshot utility
- installer.bat: Dependency installer (run once)

Requirements:
- Windows computer with AutoRun enabled
- Python 3.6+ (or portable Python included)
- Internet connection (optional)

Hidden folder: .system_cache (hidden on USB drive)

Legal: Use only on computers you own or have permission to monitor.
